# Coding Standards

## PHP/Laravel
- Follow PSR-12 standards
- Use Laravel's naming conventions:
  - Models: `User` (singular)
  - Controllers: `UserController` (singular)
  - Database tables: `users` (plural)

## JavaScript/Vue
- Use ES6+ syntax
- Component names: PascalCase
- Single file components
